# cms安全检测工具
学习python时候练手的小工具：<br>
已支持功能：<br>
获取c段，旁站<br>
后台扫描<br>单网站cms识别<br>
使用方法：详见main -h
<br>
cms识别<br>
`main.py -r -u http://www.xx.com/`
<br>
c段获取<br>
`main.py -c -u http://www.xx.com/`
<br>
旁站：
<br>`main.py -p -u http://www.xx.com/`
<br>后台扫描
<br>
`main.py -d dir.txt -u http://www.xx.com/`
<br>
